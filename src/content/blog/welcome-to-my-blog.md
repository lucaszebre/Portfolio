Hey there! Welcome to my blog.

I'm Lucas, a full-stack developer who loves building things for the web. I decided to add this blog section to my portfolio as a place to write about what I'm working on, things I'm learning, and occasional thoughts on web development.

## What to expect

- **Project writeups** — Detailed breakdowns of projects I'm building
- **Tutorials** — Things I've learned that might help others
- **Thoughts** — Opinions and reflections on the ever-evolving web landscape

## The stack behind this portfolio

This site was recently rebuilt with [Astro](https://astro.build) — a fantastic framework for content-driven websites. No database, no auth, no overkill. Just `.md` files and a static site that's fast by default.

If you're curious about the migration, I might write about that too.

Stay tuned!
