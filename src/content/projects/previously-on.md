Previously On is a web application for tracking favorite TV series and movies. Built with Next.js and the BetaSeries API, it provides a comprehensive platform for managing watched content and exploring new shows.

## Features

- **Watch List Management** — Keep track of series and movies you've watched.
- **Episode Tracking** — Mark episodes as watched or unwatched.
- **Actor Details** — Explore detailed information about actors.
- **Trailer Viewing** — Watch trailers for series and movies.
- **BetaSeries Integration** — Log in with your BetaSeries account.
- **Responsive Design** — Great experience on desktop and mobile.

## Tech Stack

- **Framework:** React.js
- **Data:** React Query + BetaSeries API
- **UI:** shadcn/ui + Tailwind CSS
