An art gallery slideshow site built to showcase paintings with slideshow navigation and lightbox viewing.

## Features

- **Slideshow Navigation** — Navigate through paintings.
- **Lightbox** — View paintings in detail with a lightbox.
- **Responsive Design** — Optimal layout for different screen sizes.
- **Hover States** — Interactive elements have hover states.

## Tech Stack

- **Frontend:** React.js
- **Data:** Local JSON file
- **Styling:** Custom CSS
