A clone of Instagram built using Next.js, Prisma, and Pusher. It replicates core features such as posting images, liking and commenting on posts, real-time updates, and user authentication.

## Features

- **User Authentication** — Sign up, log in, and log out securely.
- **Posting Images** — Upload and share images with captions.
- **Liking and Commenting** — Like and comment on posts.
- **Real-time Updates** — Pusher integration for real-time likes, comments, and new posts.
- **User Profiles** — Profile pages displaying posts and user information.

## Tech Stack

- **Framework:** Next.js
- **Database:** Prisma ORM + PostgreSQL
- **Real-time:** Pusher WebSockets
