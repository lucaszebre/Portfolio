Chat Pet is an AI-powered pet companion application developed during the WebAcademie program. Users can create virtual pets and engage in conversations with them using advanced AI technology.

## Features

- **AI-Powered Chat** — Engage in natural conversations with your virtual pet powered by Google's Gemini API.
- **User Authentication** — Secure login with GitHub and Google authentication via Better Auth.
- **Image Uploading** — Upload and share images of your pet using UploadThing integration.
- **Virtual Pet Management** — Create and customize your own virtual pet companion.
- **Responsive Design** — Access your virtual pet on both desktop and mobile devices.

## Tech Stack

- **Frontend:** Vite.js
- **Backend:** Hono on Cloudflare Workers
- **Auth:** Better Auth (GitHub + Google OAuth)
- **AI:** Google Gemini API
- **Storage:** Cloudflare D1
