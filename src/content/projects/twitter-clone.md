A Twitter clone developed using PHP, MySQL, and jQuery, replicating core features like posting tweets, following users, liking and retweeting.

## Features

- **User Authentication** — Sign up, log in, and log out securely.
- **Posting Tweets** — Compose and share tweets with text, images, or links.
- **Following Users** — Follow other users to see their tweets on your timeline.
- **Liking and Retweeting** — Like and retweet tweets from other users.
- **Real-time Updates** — jQuery AJAX for live updates.
- **User Profiles** — Profile pages with tweets, followers, and following info.

## Tech Stack

- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** jQuery
