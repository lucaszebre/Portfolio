A multi-page website for Designo, a creative design agency. Built with React.js to closely replicate the provided design.

## Features

- **Responsive Design** — Optimal layout for each page across screen sizes.
- **Hover States** — Interactive elements have hover states for better UX.
- **Contact Form Validation** — Error messages for invalid or empty fields.

## Tech Stack

- **Frontend:** React.js
- **Styling:** Custom CSS
