Linkedev is a developer's link tree web application for creating a personalized landing page with links to online profiles, projects, and social media.

## Features

- **User Authentication** — Securely log in and manage your link tree.
- **Link Management** — Add, edit, and delete links to your online profiles.
- **Customization** — Personalize with themes, colors, and backgrounds.
- **Responsive Design** — Looks great on desktop and mobile.
- **Analytics** — Track link performance and visitor statistics.

## Tech Stack

- **Frontend:** React.js + Material UI
- **Backend:** Supabase (Database + Auth)
- **Data:** React Query
