The Invoice Manager simplifies invoice management — log in, create invoices, change their status, and delete them.

## Features

- **User Authentication** — Secure login with user accounts.
- **Create Invoices** — Generate new invoices with essential details.
- **Invoice Status** — Change status (Pending, Paid, Overdue).
- **Delete Invoices** — Remove unwanted invoices.
- **Responsive Design** — Manage invoices across devices.

## Tech Stack

- **Frontend:** React.js, TypeScript, Material UI
- **Backend:** Firebase (Database + Auth)
- **State:** Redux Toolkit
