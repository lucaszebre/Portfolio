A multi-page e-commerce website for Audiophile audio products built with React.js. Browse products, manage a cart, and complete checkout.

## Features

- **Product Management** — Browse through available products.
- **Cart Management** — Add/remove products, edit quantities.
- **Checkout Process** — Fill in shipping details with form validation.
- **Checkout Totals** — Correct totals including shipping and VAT.
- **Order Confirmation** — Confirmation modal with order summary.
- **Persistent Cart** — Cart items persist via localStorage.

## Tech Stack

- **Frontend:** React.js
- **Styling:** Custom CSS
