Linkedev est une application web de link tree pour développeurs permettant de créer une page d'accueil personnalisée avec des liens vers vos profils en ligne, projets et réseaux sociaux.

## Fonctionnalités

- **Authentification utilisateur** — Connectez-vous et gérez votre link tree en toute sécurité.
- **Gestion des liens** — Ajoutez, modifiez et supprimez des liens vers vos profils en ligne.
- **Personnalisation** — Personnalisez avec des thèmes, couleurs et arrière-plans.
- **Design responsive** — Super rendu sur desktop et mobile.
- **Analytique** — Suivez les performances des liens et les statistiques de visiteurs.

## Stack technique

- **Frontend :** React.js + Material UI
- **Backend :** Supabase (Base de données + Auth)
- **Données :** React Query
