Previously On est une application web pour suivre vos séries TV et films préférés. Construite avec Next.js et l'API BetaSeries, elle offre une plateforme complète pour gérer le contenu regardé et découvrir de nouvelles séries.

## Fonctionnalités

- **Gestion de la liste de visionnage** — Suivez les séries et films que vous avez regardés.
- **Suivi des épisodes** — Marquez les épisodes comme regardés ou non regardés.
- **Détails des acteurs** — Explorez des informations détaillées sur les acteurs.
- **Visionnage de bandes-annonces** — Regardez les bandes-annonces des séries et films.
- **Intégration BetaSeries** — Connectez-vous avec votre compte BetaSeries.
- **Design responsive** — Excellente expérience sur desktop et mobile.

## Stack technique

- **Framework :** React.js
- **Données :** React Query + API BetaSeries
- **UI :** shadcn/ui + Tailwind CSS
